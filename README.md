¡Bienvenido a mi repositorio de ejercicios de lógica de programación! 🚀

## ¿Qué encontrarás aquí?

- Ejercicios de lógica de programación ✅  
- Desafíos progresivos para mejorar la lógica  💡  
- Posibles soluciones 📝
